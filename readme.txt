WordPress Responsive Gallery

== Description ==
simple but amaizing Wordpress responsive gallery.you can create gallery with mulituple images.It has greate features zoom in and zoom out , Full Screen, Slide show.

<h3>Features</h3>
<ul>
	
	<li>Touch Support</li>
	<li>Keypress Support</li>
	<li>Animated Gallery</li>
	<li>AutoPlay</li>
	<li>Turn Page by Clicking Image</li>
	<li>Navigation Tabs</li>
	<li>Arrows</li>
	<li>Zoom Images</li>
	<li>Slide Show</li>
	<li>View All</li>
</ul>

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Pr responsive' and install it
3. Go to Dashboard > pr responsive and upload images as gallery
4. Use this shortcode [pr-responsive-gallery] in any page

5. Now visit that page
